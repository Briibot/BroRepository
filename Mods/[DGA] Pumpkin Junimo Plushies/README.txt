RoseDryad's Pumpkin Junimo Plushies Mod for Stardew Valley

Author: RoseDryad
Version: 1.0.0
Description: Adds Pumpkin Junimo Plushies using Dynamic Game Assets

Requirements:
 - SMAPI (https://smapi.io/)
 - Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915)
 - Dynamic Game Assets (https://www.nexusmods.com/stardewvalley/mods/9365)
 - SpaceCore (https://www.nexusmods.com/stardewvalley/mods/1348)

Issues:
- Decorations placed on tables turn into chairs.
- Not compatiable with: CJB Item Spawner and stacking mods.

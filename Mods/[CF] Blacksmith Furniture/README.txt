*Blacksmith Furniture Mod for Stardew Valley*

Author: SpottedDotted
Version: 1.0.0
Description: Addition of blacksmith themed furniture to the game

Requirements:
 - SMAPI (https://smapi.io/)
 - Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915)
 - PyTK (https://www.nexusmods.com/stardewvalley/mods/1726)
 - Custom Furniture (https://www.nexusmods.com/stardewvalley/mods/12540)

Installation:
 - Download the latest version of all the above mods and unzip them into your game folder (Stardew Valley/Mods)
 - Download and unzip this mod into Stardew Valley/Mods
 - Run the game through SMAPI
